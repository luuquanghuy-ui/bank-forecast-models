# 3.3 Conclusions

## Executive Summary

This thesis demonstrates that:

1. **Volatility IS predictable** using XGBoost (tree-based captures clustering)
2. **Price is NOT predictable** due to martingale property - only Hybrid beats Naive
3. **Simple models beat complex ones** on financial time series with small sample size
4. **Hybrid (GARCH+Ridge)** is optimal for price prediction

---

## Key Findings

### Finding 1: Volatility Clustering is Exploitable

**Evidence:**
- XGBoost α+β ≈ 0.97 for all banks (volatility persistence)
- XGBoost beats Naive by 9-29% for volatility
- Hybrid beats Naive by 17-30% for volatility
- All improvements are statistically significant

**Implication:**
- Vietnam bank stocks exhibit strong volatility clustering
- This persistence can be exploited for risk management
- GARCH-family models capture this structure well

---

### Finding 2: Price Follows Random Walk

**Evidence:**
- Naive beats NeuralProphet by 90-185%
- Naive beats TFT by 176-1433%
- Complex models perform WORSE than predicting today's price

**Implication:**
- Stock prices follow martingale process
- No patterns to exploit in short-term price movements
- Complex models overfit noise and perform worse

---

### Finding 3: Complexity Hurts Performance

**Evidence:**

| Model | Complexity | vs Naive (Price) | vs Naive (Vol) |
|-------|-----------|-------------------|----------------|
| XGBoost | ~50 params | -251-510% | **+9-29%** |
| NeuralProphet | ~1000 params | -186-210% | N/A |
| TFT | ~10000+ params | -176-1433% | N/A |
| Hybrid | ~20 params | **+35-41%** | +17-30% |

**Implication:**
- Financial time series have low signal-to-noise ratio
- More parameters = more overfitting
- Simple models (XGBoost, Hybrid) work best for their targets

---

### Finding 4: Hybrid Ensemble is Optimal (for Price)

**Evidence:**
- Hybrid beats Naive by 35-41% for price
- Hybrid beats XGBoost by 80-90% for price (XGBoost severely overfits)
- Best weight ≈ 0.45 (roughly equal contribution from GARCH signal + Ridge)
- All improvements statistically significant

**Implication:**
- For volatility: XGBoost is optimal (tree-based captures clustering)
- For price: Hybrid (GARCH+Ridge) is optimal (stable, no overfitting)
- Each model has its target where it excels

---

## Thesis Contributions

### 1. Demonstrated Martingale in Vietnam Banks
First study to show price unpredictability in Vietnam commercial bank stocks.

### 2. Quantified Volatility Clustering
Showed α+β ≈ 0.97, higher than typical developed markets.

### 3. Proposed Hybrid Model
GARCH + Ridge ensemble that beats either component alone.

### 4. Comprehensive Evaluation
Used 4-fold walk-forward, DM tests, and sensitivity analysis.

## Practical Applications

### For Risk Management
- Use Hybrid model to forecast daily volatility
- Calculate VaR (Value at Risk) using forecasted volatility
- Allocate capital based on volatility predictions

### For Trading
- Volatility forecasts can inform options pricing
- High volatility periods = wider stop-losses
- Low volatility periods = tighter risk controls

### For Academic Research
- Framework for comparing price vs volatility models
- Methodology for fair model comparison
- Evidence on complexity vs simplicity in financial forecasting

---

## Limitations

### 1. Sample Size
- Only ~1750 observations per bank
- Limits complexity of applicable models
- Larger dataset might benefit deep learning

### 2. Single Market
- Only Vietnam commercial banks
- Results may not generalize to other markets
- Different volatility dynamics in developed markets

### 3. No Macroeconomic Features
- Model uses only price/volume data
- Could be improved with fundamental data
- News sentiment not included

---

## Final Recommendation

**For volatility prediction:**
```
USE: XGBoost (tree-based captures volatility clustering)
NOT: NeuralProphet, TFT (wrong target, overfit)
```

**For price prediction:**
```
USE: Hybrid (GARCH volatility signal + Ridge regression)
NOT: XGBoost, NeuralProphet, TFT (all overfit badly)
```

---

## Closing Statement

This thesis demonstrates that in financial time series forecasting:

> **"Simpler is better, and the right model must match the right target."**

- Volatility clustering makes volatility predictable → Use XGBoost
- Martingale property makes price unpredictable → Use Hybrid (only model beating Naive)
- Complexity overfits noise → Use simple models
- Each target needs its own optimal model
