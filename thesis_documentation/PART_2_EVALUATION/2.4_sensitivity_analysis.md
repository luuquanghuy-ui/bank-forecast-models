## Summary of Sensitivity Analysis

| Model | Parameter | Finding |
|-------|-----------|---------|
| GARCH | Split ratio | Robust - beats Naive at all ratios |
| NeuralProphet | Learning rate | Lower is better (less overfitting) |
| NeuralProphet | Epochs | Fewer is better (less overfitting) |
| TFT | Hidden size | Smaller is better (less overfitting) |
| TFT | Attention heads | Fewer is better (less overfitting) |
| TFT | Epochs | Fewer is better (less overfitting) |
| Hybrid | Ensemble weight | ~0.45 optimal |
| Hybrid | Ridge alpha | 1.0 optimal |

---

## Key Insight

**All deep learning models (NP, TFT) suffer from overfitting.**

The pattern is clear:
- More parameters = more overfitting
- More epochs = more overfitting
- Higher learning rate = more overfitting

This confirms that financial time series have low signal-to-noise ratio, making complex models unsuitable.
