# 2.3 Statistical Tests

## Diebold-Mariano Test

The Diebold-Mariano (DM) test determines if one forecast is significantly better than another.

### Null Hypothesis
```
H0: Forecasts have equal accuracy
```

### Interpretation
- p < 0.05: Reject H0 → Significant difference
- p < 0.01: Highly significant (***)
- p < 0.05: Significant (*)

## Residual Analysis: ACF and PACF

### Why Check Residuals?

After fitting GARCH, residuals should be:
- Uncorrelated (no autocorrelation)
- Normally distributed
- Heteroskedasticity removed

### ACF (Autocorrelation Function)

ACF measures correlation between lagged values.

**Good residual ACF:** All bars within confidence bands (no significant correlations)

### PACF (Partial Autocorrelation Function)

PACF measures direct correlation at each lag, controlling for intermediate lags.

**Good residual PACF:** No significant spikes after lag 0

## Market Event Validation

### What is Market Event Validation?

Test model performance on extreme volatility days:
- Days with highest absolute returns
- Days surrounding market stress events

### Purpose

Ensure models work when it matters most:
- High volatility = when forecasts are most valuable
- Stress periods = when accurate risk assessment is critical
