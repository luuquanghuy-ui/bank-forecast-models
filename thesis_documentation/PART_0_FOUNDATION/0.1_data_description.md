# 0.1 Data Description

## Data Source

**Vietnam Commercial Bank Stock Prices (2016-2026)**

Three major Vietnamese banks:
- **BID**: BIDV (Bank for Investment and Development of Vietnam)
- **CTG**: VietinBank (Vietnam Joint Stock Commercial Bank)
- **VCB**: Vietcombank (Joint Stock Commercial Bank for Foreign Trade of Vietnam)

## Original Dataset

- **File:** `banks_BID_dataset.csv`, `banks_CTG_dataset.csv`, `banks_VCB_dataset.csv`
- **Columns (17):** index, date, open, high, low, close, volume, ticker, vnindex_close, vn30_close, log_return, volatility_20d, ma20, ma50, rsi, usd_vnd, interest_rate
- **Time Range:** 2016-02-04 to 2026-02-27

## Preprocessing Steps

### 1. Data Collection
Data collected from three sources:
- **Yahoo Finance**: OHLCV prices for 3 bank tickers
- **Investing.com**: VNIndex, VN30 daily closes
- **Ngân Hàng Nhà Nước VN**: USD/VND exchange rate, interbank interest rate

### 2. Feature Engineering

| Column | Formula | Description |
|--------|---------|-------------|
| `date` | - | Datetime index |
| `open/high/low/close` | - | OHLC prices |
| `volume` | - | Trading volume |
| `vnindex_close` | - | VNIndex closing price |
| `vn30_close` | - | VN30 closing price |
| `log_return` | log(close_t / close_{t-1}) | Log return |
| `volatility_20d` | std(log_return_{t-19:t}) | 20-day volatility |
| `ma20` | mean(close_{t-19:t}) | 20-day moving average |
| `ma50` | mean(close_{t-49:t}) | 50-day moving average |
| `rsi` | RSI(14) | Relative Strength Index |
| `usd_vnd` | - | USD/VND exchange rate |
| `interest_rate` | - | Interbank interest rate (%) |

### 3. Data Split (70/15/15)

| Dataset | Purpose | Description |
|---------|---------|-------------|
| Train | 70% | Model training |
| Validation | 15% | Hyperparameter tuning |
| Test | 15% | Final evaluation |

### 4. Target Variables

Two different targets depending on model:

| Target | Scale | Models |
|--------|-------|--------|
| Price | ~40-60 VND | NeuralProphet, TFT, Naive |
| Volatility | ~0.01 | GARCH, Hybrid, Ridge, Random Forest |

## Key Characteristics

### Volatility Clustering
High volatility today → likely high volatility tomorrow

Evidence from GARCH parameters:
| Bank | α (ARCH) | β (GARCH) | α+β |
|------|---------|----------|-----|
| BID | 0.059 | 0.920 | 0.979 |
| CTG | 0.090 | 0.888 | 0.978 |
| VCB | 0.076 | 0.886 | 0.962 |

**α+β ≈ 0.97** indicates high volatility persistence → volatility IS predictable

### Price Random Walk
Stock prices follow martingale process:
E[S_{t+1} | I_t] = S_t

Evidence: Complex models (NP, TFT) perform WORSE than naive prediction (predict today's price)
