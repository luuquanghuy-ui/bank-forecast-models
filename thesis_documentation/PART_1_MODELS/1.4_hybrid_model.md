# 1.4 Hybrid GARCH+Ridge Model

## Model Description

**Hybrid** = GARCH + Ridge Regression Ensemble

This is the **BEST overall model** in our thesis. It combines:

1. **GARCH(1,1)** - Econometric model capturing volatility clustering
2. **Ridge Regression** - ML model capturing short-term patterns

### Equation

```
Hybrid_pred = w × GARCH_pred + (1-w) × Ridge_pred
```

Where `w` is the ensemble weight optimized on validation data.

---

## Why Hybrid Works Better Than Either Alone

### GARCH Strengths
- Captures long-term volatility persistence
- Theoretically grounded
- Works well in volatile periods

### Ridge Strengths
- Captures short-term patterns from ML features
- Fast and interpretable
- Works well in calm periods

### Hybrid: Best of Both Worlds
- Adaptive to market conditions
- GARCH + Ridge compensate for each other's weaknesses

## Why Hybrid is Better Than GARCH Alone

### 1. Ensemble Effect
- GARCH captures volatility structure
- Ridge captures short-term anomalies
- Combined: handles both calm and volatile periods

### 2. Adaptive Weight
- Best weight found on validation set
- ~0.45 means roughly equal contribution
- Neither model dominates

### 3. Lower Variance
- Ensemble reduces prediction variance
- More robust to regime changes
