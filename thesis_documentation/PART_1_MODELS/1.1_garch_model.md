# 1.1 GARCH Model

## Model Description

**GARCH(1,1)** - Generalized Autoregressive Conditional Heteroskedasticity

GARCH models volatility as time-varying and dependent on past shocks.

> **IMPORTANT:** GARCH is NOT a standalone model in the 4-fold comparison. GARCH is used WITHIN the Hybrid model to provide volatility signal. This document explains WHY GARCH was incorporated into Hybrid.

### Equation

```
σ²_t = ω + α·ε²_{t-1} + β·σ²_{t-1}
```

| Parameter | Name | Meaning |
|-----------|------|---------|
| ω (omega) | Constant | Base variance |
| α (alpha) | ARCH term | Impact of past shock ε²_{t-1} on today's variance |
| β (beta) | GARCH term | Impact of past variance σ²_{t-1} on today's variance |

### Volatility Persistence

**α + β** close to 1 means:
- Volatility shocks decay slowly
- High volatility today → likely high tomorrow
- This is WHY volatility is predictable!
- This is WHY Hybrid was developed using GARCH signal!

---

## Why GARCH is in Hybrid

### 1. GARCH Proves Volatility Clustering
- α+β ≈ 0.97 for all banks → strong persistence
- This proves volatility IS predictable
- XGBoost wins volatility in 4-fold CV by capturing this

### 2. GARCH Provides Signal to Hybrid
- GARCH volatility forecast is an INPUT to Hybrid
- Hybrid uses: GARCH_signal × w + Ridge_prediction × (1-w)
- This is the thesis contribution: combining econometric + ML

### 3. Theoretical Foundation
- GARCH is the standard in financial econometrics (Engle 1982, Bollerslev 1986)
- Parameters have economic interpretation
- GARCH signal is well-theorized, not a black box

## How GARCH is Used in Hybrid

GARCH is NOT compared standalone. Instead:

```
Hybrid_pred_vol = w × GARCH_vol + (1-w) × Ridge_features
```

Where:
- GARCH_vol = GARCH(1,1) volatility forecast
- Ridge_features = lagged returns, RSI, volume, etc.
- w ≈ 0.45 (optimized on validation set)

---

## Why Not GARCH Alone?

1. **GARCH alone doesn't beat XGBoost** on 4-fold CV
2. **Hybrid adds Ridge features** for additional signal
3. **For price prediction**, GARCH alone is insufficient - Hybrid (GARCH+Ridge) is needed

**Key result:** XGBoost wins volatility (not GARCH standalone), Hybrid wins price.
