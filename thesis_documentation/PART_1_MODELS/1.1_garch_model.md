# 1.1 GARCH Model

## Model Description

**GARCH(1,1)** - Generalized Autoregressive Conditional Heteroskedasticity

GARCH models volatility as time-varying and dependent on past shocks.

### Equation

```
σ²_t = ω + α·ε²_{t-1} + β·σ²_{t-1}
```

| Parameter | Name | Meaning |
|-----------|------|---------|
| ω (omega) | Constant | Base variance |
| α (alpha) | ARCH term | Impact of past shock |ε²_{t-1} | on today's variance |
| β (beta) | GARCH term | Impact of past variance |σ²_{t-1} | on today's variance |

### Volatility Persistence

**α + β** close to 1 means:
- Volatility shocks decay slowly
- High volatility today → likely high tomorrow
- This is WHY volatility is predictable!

---

## Why GARCH is Appropriate

### 1. Financial Time Series Properties
- Returns have time-varying volatility
- Volatility clusters (high vol today → high vol tomorrow)
- Past information affects current volatility

### 2. Theoretical Foundation
GARCH is the standard in financial econometrics since Engle (1982) and Bollerslev (1986).

### 3. Interpretability
Parameters have clear economic meaning:
- α = short-term shock response
- β = long-term volatility level

## EGARCH and GJR-GARCH Variants

### EGARCH (Exponential GARCH)

Captures asymmetric volatility (leverage effect):
```
log(σ²_t) = ω + α·|ε_{t-1}/σ_{t-1}| + γ·ε_{t-1}/σ_{t-1} + β·log(σ²_{t-1})
```

### GJR-GARCH (Glosten-Jagannathan-Runkle)

Also captures leverage effect:
```
σ²_t = ω + (α + γ·I_{t-1})·ε²_{t-1} + β·σ²_{t-1}
```

Where I_{t-1} = 1 if ε_{t-1} < 0 (negative shock)

**All variants perform similarly. GARCH(1,1) is sufficient.**
